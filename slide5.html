toggleライブラリの応用

$('.toggle').toggles()に設定するプロパティ一覧

		
		$('.toggle').toggles({
		drag : true,
		click : true,
		text : {
		  on : 'ON',
		  off : 'OFF'
		},
		on : true,
		animate : 250,
		transition : 'ease-in-out',
		checkbox : null,
		clicker : null,
		width : 50,
		height : 20,
		type : 'compact'
		});
